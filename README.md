# Crixsoft-Solution-_-Project_HEART-DISEASE-DETECTION
Develops multiple machine learning models to classify patients as having heart disease or not based on medical data. Models include Logistic Regression, Random Forest, Gradient Boosting, and SVM.
PROJECT 2: Heart Disease Detection 💓

File: Project_2_Heart_Disease_Detection.py

What it does:


Classifies patients as having heart disease or not
Uses 4 different ML algorithms for comparison
Provides risk assessment for individual patients


Key Features:


✅ Medical dataset generation
✅ 4 classification models (Logistic Regression, Random Forest, Gradient Boosting, SVM)
✅ Cross-validation evaluation
✅ ROC curve analysis
✅ Patient risk prediction
✅ Comprehensive visualizations


Output:

Best Model: Random Forest
Accuracy: 87.5%
Precision: 0.85
Recall: 0.82
F1-Score: 0.83
Sample Prediction: Patient has 58% risk of heart disease
Visualization: Heart_Disease_Detection_Analysis.png

Time to Run: ~3-4 minutes
